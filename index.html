学Swift还在写五子棋？不如来玩多边形的寓言

发表于 2017-02-23  |  2017-02-23

简介

一个纯Swift写的iOS练手的寓言小游戏，并上架AppStore

Github仓库：点我

下载地址：点我

复杂效果视频演示： 点我

Idea来自于一个PolygonStory的网页（原作地址：点我），做出来的效果与原作效果基本一致，但由于只懂iOS的内容，所以与原作的HTML代码应该会有很大差别。

这次项目的学习点：

  • 数据驱动视图的设计思路

  • 动态走势图的绘制（贝塞尔曲线+定时器）

  • 类棋盘的触摸板二维界面布置

  • 实现触摸手势的综合复杂判断

    • 需要结合

      • 当前多边形与其周边同伴的心情
      • 多边形的位置（起始点与终点）
      • 手势状态（start/change/end）

    • 触摸手势划出设定边界的判断问题

  • 多边形自行完成游戏（自己选点，自己做判断，每个多边形会根据周边的情况实时做出反应）

    • 通过让多边形自我移动的动画时间远超定时器从而达到多个多边形同时移动的假象

  • 通过两张图的视差达到按钮点击下沉的动画效果

  • 富文本的基本使用

以上可见：其实我并没有使用很多复杂的控件，更多的是逻辑上的设计，自己还是蛮喜欢完这种小逻辑的。

游乐园定理

发表于 2016-11-17  |  2016-11-17   |   分类于 ORID日记

Objective (客观描述事实)

Facts, Data, Senses

  • 安全区小,周边环境关联性低 -> 超高效率
  • 很久没有看手机,看新闻 -> 超高效率
  • 晚上会看会儿youku,youtubu -> 超低效率

Reflective (主观描述感想)

Reactions, Heart, Feelings

  • 一觉醒来,才发现,昨天晚上沾床就睡…对于一个贪睡的人…唉…又丢失了一个晚上的时间,至少睡前看点题嘛..也没有…好桑心…

Interpretative (分析寻找意义何在)

So What?

  • 晚上的时间利用率一般都比较低,而一般普遍认为晚上的时间还是蛮宝贵的,因为像大部分人不论是工作还是上学,都是如此,晚上都是自由安排的时间,在一天工作即将结束之际,夜晚降临之前,一般像我这样的人,就会开始幻想漫漫长夜,我要如何如何好好利用这段宝贵的夜晚时间,结果到了那个点…刷刷手机上上网,聊个天,时间就过了大半了,再有像我这样,贪睡的,说先咪一会儿,再起来看书,起来学,结果从来都是,一觉睡到大天亮…桑心啊…
  • 经过这么多年了,可以总结下自己是什么人
    • 无法接收干扰,哪怕一点点(这里的干扰指的是,碎片化信息,视频,电影,聊天,等等…)
    • 自己是个爱睡觉的人
    • 自己在公共场所的安全距离比较小,很容易集中精力,但是,如果有一个手机可以干一些不相干的事情,那么也就完蛋了
  • 综上所述,我要面对的就是如何抛弃那些没用的信息,或者说是,如何阻止那些会干扰我集中精力的信息进入我的世界,我应该想办法,让所有这些新闻,视频,都远离自己的正常生活,这些被排斥的东西,应该以游乐园的形式,定期出现在我的生活里,起到调剂性的作用就好,也许两个星期感觉会比较合适
    • 难以想象,一个由互联网的到思想解放的人,现在正在想各种办法,刨去掉自己喜爱的以大部分,竟然只是为了集中精力,不过想象,以前即使没有互联网,人一样很难集中精力,还会有别的很多干扰,别的很多娱乐方式…所以,我要 把这些娱乐的东西刨开来,统一放在”游乐园”里,定期去逛一逛就好了.

Decision (决定下一步的行动)

Now What?

  • ​想一个比较好的”游乐园”点子
